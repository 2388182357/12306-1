12306 订票助手(谷歌浏览器版)
===========================

请使用谷歌浏览器

功能
------------

*	自动尝试登录，登录成功提示
*	根据条件自动查票，有票以后声音提示


使用
------------

有使用 12306自动登录 和 12306自动查询 的同学可在[插件管理](chrome://settings/extensions) 里面卸载了，因为功能重复了

使用谷歌浏览器 [点击这里](), 然后确认安装userscript

安装成功后 [点击这里](https://dynamic.12306.cn/otsweb/) 开始订票吧
